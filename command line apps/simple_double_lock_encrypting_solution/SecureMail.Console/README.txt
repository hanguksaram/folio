Заготовка программы к задаче про ромео и джульету
-----
Приложение эмулирует передачу сообщения между двумя адресатами.
Текущая реализация (NotSecureMailStrategy.cs) не защищает сообщение от почтальона.
Необходимо внести изменение в файл SecureMailStrategy.cs таким образом чтобы была реализована защищенная передача сообщения, остальные части кода менять не нужно.
Для проверки кода, используйте Program.cs, в котором эмулирована передача сообщениям между Ромео и Джульетой.

-----
Параллельно с выполнением задания, ознакомьтесь со структурой кода - Dependency Injection, Factory pattern, Strategy pattern, interfaces, etc.
